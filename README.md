# Word-counter
Counts words in .docx files.
Node.js is required to build and run this project.

# Initialization
```
git clone https://github.com/nozols/Word-counter.git
```
```
npm install
```

# usage
```
node index.js
```

## parameters
```
node index.js --excel=filename.xlsx
```
```
node index.js --json=json.txt
```
